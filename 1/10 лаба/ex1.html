Flexbox (Flexible Box Layout) — это мощный инструмент CSS для создания гибких и адаптивных макетов. Он позволяет легко управлять пространством между элементами и их выравниванием в одномерной плоскости — либо по горизонтали, либо по вертикали. Основные концепции Flexbox Flex-контейнер и Flex-элементы: Flex-контейнер — это родительский элемент, на который применяется свойство display: flex;. Flex-элементы — это дочерние элементы внутри flex-контейнера. Главная и поперечная оси: Главная ось (main axis) определяется свойством flex-direction и может быть горизонтальной (row) или вертикальной (column). Поперечная ось (cross axis) проходит перпендикулярно главной оси. Основные свойства Flexbox: flex-direction: определяет направление главной оси (row, row-reverse, column, column-reverse). justify-content: выравнивает элементы вдоль главной оси (flex-start, flex-end, center, space-between, space-around). align-items: выравнивает элементы вдоль поперечной оси (flex-start, flex-end, center, stretch, baseline). flex-wrap: управляет переносом элементов на новую строку (nowrap, wrap, wrap-reverse). align-content: управляет выравниванием нескольких строк flex-элементов (flex-start, flex-end, center, space-between, space-around, stretch).
